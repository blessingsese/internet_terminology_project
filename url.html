Uniform Resources Locator

A URL is the specific address of a resource on the web.
https://www.example.com:80/path/to/page.html?query=1
Scheme (Protocol): https:// — Tells the browser how to connect.
Subdomain: www. — A subdivision of the main domain.
Domain Name: example.com — The specific name of the website.
Port: :80 — The technical "gate" used to access the server.
Path: /path/to/page.html — The specific file or location on the server.
Query String: ?query=1 — Parameters used to provide extra data to the server.
